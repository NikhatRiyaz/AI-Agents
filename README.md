# Agentic Honey-Pot Scam Detection & Intelligence Extraction

A Python-based AI system designed to detect scams, classify messages, and extract intelligence using agentic workflows and state graphs.

---

## Features

- **Scam Detection**: Classifies messages, emails, or text as scam or non-scam.
- **Intelligence Extraction**: Extracts key information from scam attempts.
- **Agentic Architecture**: Modular design using state graphs for flexible workflow automation.
- **API-Ready**: Easily deployable via REST API.
- **Docker Support**: Containerized setup for quick deployment.
